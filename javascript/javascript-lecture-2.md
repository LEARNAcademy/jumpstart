## Math.floor()
Rounds down to the nearest whole number
```javascript

```

## Math.Random()
Generates a random number between 0 and 0.9
** First show it by itself **
```javascript
console.log(Math.random())
```
** Then with * 3 **
```javascript
console.log(Math.random() * 3)
```

- combined we get:
```javascript
console.log(Math.floor(Math.Random() * 3))
```
